Welcome to the Brooklyn Community Catalog

This website lists all Apache Brooklyn entities that users make available to the outside world. You can see all the relevant information about a blueprint directly onto the detail pages.

Here is how you can use these blueprint:

  1. Select the blueprint that you are interested in
  2. Copy the catalog YAML blueprint
  3. Paste it into your Brooklyn catalog
  4. You're good to go!
